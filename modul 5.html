Konsep

Forthes
Forthes adalah sistem pembuka file yang aman untuk semua orang dan meminimalisir peretasan via file. Sistem ini memiliki 3 zona:
  1. Zona pertama adalah zona aman untuk masyarakat umum. Segala bentuk script akan dibersihkan agar terhindar dari serangan zero-day.
  2. Zona kedua lebih fleksibel, hanya kode terverifikasi yang bisa berjalan, seperti perhitungan akuntansi.
  3. Zona ketiga untuk developer, memungkinkan menjalankan kode bebas namun diawasi ketat oleh Cognitive Sentinel.

Cognitive Sentinel
Cognitive Sentinel adalah pengawas perilaku secara real time, memantau aktivitas pegawai berdasarkan skor kepercayaan.
Nilai Kepercayaan Tindakan
100-90 Aman
90-50 Verifikasi ulang OTP via ponsel
50-30 Verifikasi oleh tim keamanan digital
30-0 Pemblokiran total

AWS
AWS atau Adaptive Spider Web adalah konsep penempatan noise pada kamera untuk mengacaukan output AI generatif, sekaligus bisa berfungsi sebagai pola verifikasi.

Cerberum
Cerberum adalah perangkat IoT yang berfungsi menjaga perangkat lain agar lebih aman, dan membantu mengintegrasikan perangkat dari merek yang berbeda.
